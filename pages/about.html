About MTGO Set Usage

This site tracks which cards from each Magic: The Gathering set see play in competitive MTGO events, broken down by format (Standard, Pioneer, Modern, Legacy, Vintage, Pauper).

Decklist data is sourced from the MTGO official decklist page. Card information is provided by the Scryfall API. Standard rotation data is provided by whatsinstandard.com.

Source code is available on GitHub.
